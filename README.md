# COVID-19 Data Analysis Project

This project analyzes COVID-19 data using Python libraries such as **Pandas**, **NumPy**, **Matplotlib**, and **Plotly Express**. The goal is to understand the COVID situation across different countries through data cleaning, manipulation, and visualizations.

 Features of the Project
- Data loading and cleaning  
- Data manipulation using Pandas  
- Interactive visualizations using Plotly  
- Static graphs using Matplotlib  
- Country-wise COVID-19 comparison  
- Trend analysis and insights

Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Plotly Express  
- Google Colab Notebook

Visualizations Included
- Line charts  
- Bar graphs  
- Scatter plots  
- Country-wise comparisons  
- Trend visualizations  

 Dataset
The project uses a dataset named **covid.csv**, containing:
- Country  
- Confirmed Cases  
- Recovered Cases  
- Deaths  
- Active Cases  
- Date-wise data  


How to Run This Project
1. Download or clone this repository  
2. Open the `.ipynb` notebook in Google Colab or Jupyter Notebook 
3. Upload your `covid.csv` file  
4. Run the notebook cells one by one
